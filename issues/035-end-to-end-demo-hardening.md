## What to build

Perform an end-to-end demo hardening pass across ingestion, retrieval, strict answering, source previews, tracing, evaluation, and UI workflows using representative local public PDFs.

## Acceptance criteria

- [ ] A maintainer can ingest local public PDFs and see document status.
- [ ] A technician can ask grounded, partial, and not-found questions through the UI.
- [ ] Citations and source previews are inspectable for supported answers.
- [ ] Evaluation can run and report meaningful scores.
- [ ] Remaining prototype risks, manual setup steps, and follow-up issues are documented.
